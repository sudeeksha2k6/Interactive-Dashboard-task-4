# Interactive-Dashboard-task-4
This dashboard provides an overview of overall sales performance, profitability, and trends over time.Â Use the filters to analyse performance by year and region.
The objective is to design an interactive dashboard for business stakeholders.
Tools used: Tableau, Microsoft Excel for Data, Canva for PPT.
**Key Features: **
KPIs used: Profit, Sales, Profit Margin
Sales trend over time(Line chart)
Regional Performance, including the Regional map
Category Analysis: Bar charts compare sales and profit across product categories to:
1. Identify the most profitable categories
2. Highlight underperforming segments

Interactive Filters: Region filter, Year of Date filter
Key Insights:
1. The West region generates the highest sales.
2. The technology category shows the highest profit margin.
3. Sales show steady growth with seasonal spikes.
